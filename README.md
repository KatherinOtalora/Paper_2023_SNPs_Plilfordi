# Population genetics and phylogeographic history of the insular lizard *Podarcis lilfordi*

This repository includes several codes for the identification of SNPs from GBS data, as well as the incorporation of SNPs from RAD-seq data reported by Bassitta et al. (2021). It also contains tools for downstream analyses, including genetic structure, phylogeographic patterns, diversity indices, and outlier detection.
